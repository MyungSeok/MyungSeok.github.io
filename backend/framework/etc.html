# Framework vs Library

가장 크게는 제어의 흐름이 다르며
Framework 는 사용자의 코드가 프레임워크에 사용되는 것이고
Library 는 사용자에 코드에 의해 소비되는 것이다.

Framework 는 IoC 개념이 적용되어야 하며 이는 특정의 프로세스를 프레임워크가 처리 가능하도록 제어를 넘김으로써 클라이언트 코드가 라이브러리의 메소드를 호출해서 사용하는 것 을 의미합니다.
