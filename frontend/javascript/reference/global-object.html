# Null

변수를 선언하고 null 객체를 삽입한 상태 (객체의 참조 연결이 해지된 것)
자료형은 object 로 분류됨

console.log(typeof null);     // "object"

# Undefined

변수 선언만 하고 값을 할당하지 않는 상태 (정의가 되지 않는 변수)
자료형이 결정되지 않는 상태
