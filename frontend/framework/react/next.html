# Next.js

서버랜더링을 쉽게 구현 할 수 있도록 도와주는 프레임워크의 일종이다.

대표적인 기능으로는 다음과 같다.

  • 클라이언트 페이지 라우팅
  • HMR (Hot Moudle Replacement 을 지원하는 Webpack 기반 작업 환경)

react-router 나 react-hot-loader, redux 등을 대부분의 어려운 세팅 작업을 쉽게 적용 가능하게 도와준다.
