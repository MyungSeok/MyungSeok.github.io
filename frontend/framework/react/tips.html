# Tips

# ESLint

.eslintrc.json 파일에 다음과 같이 추가해준다.
es6 문법과 jsx 를 혼용하기 때문에 아래와 같은 lint 설정을 추가해준다.

{
    ...
    "parserOptions": {
        "ecmaVersion": 6,
        "sourceType": "module",
        "ecmaFeatures": {
            "jsx": true
        }
    }
    ...
}
