# 스택

  • 마지막에 저장한 데이터를 가장 먼저 꺼내는 후입선출 (LIFO : Last In First Out) 구조로 되어 있다.
  • 입력과 출력이 모두 한 방향에서 이루어지는 데이터 구조

# 단점

  • 바닥에 있는 데이터가 잔류하게 되는 경우가 생겨 우선순위에 관련된 문제가 생길 수 있다.
